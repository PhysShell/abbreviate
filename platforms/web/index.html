abbreviate демо

Пишите сокращение без гласных — движок предлагает полное слово. Примеры: првт, тстрние, сгдня, рбте.
Без мыши: 1…9 — вставить вариант по номеру; Tab — войти в список, ←/→ — между вариантами, ↓ — формы слова (падежи по предложению), Enter — вставить, Esc — назад. Мышь тоже работает. Всё офлайн в браузере; ничего не отправляется на сервер.
